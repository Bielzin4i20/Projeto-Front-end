Açaí do Cerrado – Site Institucional

Este repositório contém o código-fonte do site institucional do Açaí do Cerrado, desenvolvido como parte de um projeto acadêmico. O objetivo do site é apresentar a empresa, seu cardápio, informações de contato e páginas complementares, garantindo uma estrutura organizada e responsiva para publicação no GitHub Pages.

O site foi projetado para ser intuitivo, leve e visualmente atraente, com foco em apresentar produtos e informações essenciais aos clientes. A navegação foi estruturada em cinco páginas principais, cada uma com conteúdo específico, garantindo uma experiência de usuário clara e organizada.

Estrutura do Projeto

A estrutura base do projeto é composta pelos seguintes arquivos e diretórios:

/
|-- index.html
|-- sobre.html
|-- cardapio.html
|-- inverno.html
|-- contato.html
|-- style.css
|-- script.js
|-- /img
|     └── (imagens utilizadas no site)
|-- README.md

Descrição de cada arquivo

index.html
Página inicial do site, contendo destaque visual, apresentação geral e links para as demais páginas.

sobre.html
Página detalhada sobre a história da empresa, valores, missão e visão.

cardapio.html
Página completa contendo os itens do cardápio, com descrições, categorias e preços.

inverno.html
Página especial com o cardápio sazonal do inverno, incluindo bebidas quentes, sobremesas e combos.

contato.html
Página com formulário de contato, informações da empresa e mapa integrado.

style.css
Arquivo de estilos responsável pela identidade visual do site.

script.js
Arquivo JavaScript com funções de interatividade, como carrossel, animações, menu mobile e modais.

/img
Pasta contendo imagens utilizadas nas páginas (banners, fotos de produtos, ícones e demais ilustrações).

Como Executar o Projeto Localmente

Baixe ou clone este repositório:

git clone https://github.com/SEU-USUARIO/NOME-DO-REPOSITORIO.git


Abra a pasta do projeto.

Dê um duplo clique em qualquer arquivo .html para abrir no navegador.

Certifique-se de manter a estrutura de diretórios exatamente como no repositório, especialmente a pasta img e os arquivos style.css e script.js.

Publicação no GitHub Pages

O site foi preparado para funcionar no GitHub Pages.

Para publicar:

Acesse o repositório no GitHub.

Vá em Settings.

Selecione Pages no menu lateral.

Em "Source", escolha a opção:

Deploy from a branch


Selecione o branch:

main


E a pasta:

root


Salve as alterações.

Após alguns minutos, o GitHub Pages gerará um link no formato:

https://seu-usuario.github.io/nome-do-repositorio/


Use esse link para visualizar e compartilhar o site.

Tecnologias Utilizadas

HTML5 para estruturação das páginas.

CSS3 para estilização e responsividade.

JavaScript para interações no front-end (carrossel, animações, menu mobile e modal).

GitHub Pages para hospedagem gratuita do projeto.

Font Awesome para ícones visuais.

Objetivo Educacional

Este projeto foi desenvolvido para fins acadêmicos, com o objetivo de aplicar conceitos fundamentais de:

Estruturação de sites com múltiplas páginas.

Componentização visual utilizando CSS.

Interatividade com JavaScript.

Hospedagem estática via GitHub Pages.
